SEO: stands for Search Engine Optimization, it's the process of increasing the quality and quantity of website traffic, by creating or editing a website that Google can understand better and therefore it'll help you to rank#1 for a search term. Also the title and meta description are going to appear when people search for someting in a browser.

There are two main subcategories:

Off-page SEO: niche of the website, outside links pointing to the website, high quality content.

On-page SEO: Aspects that can be implemented directly by the developer.

Title

title tag should be 50-60 characters in length.

title tag are huge signal in helping search engines understand what your page is about.

The're the most important on-page SEO ranking factor and should contain the main keywords that your page is targeting.

Description

Should be unique and should contain the primary keyword.

Character length must be under 156.

URL

url needs to explain what the website is about.

It needs to be easy to understand, and tell search engine engines, and humas what the page is about.

should be under 115 characters, but the shorter the url the better.

Favicon

The main reason your website should have a favicon is improving user experience.

it is much easier to find the needed bookmark by image, associated with a certain website, rather than looking through different urls

Keyword usage

Keywords must appear in title, Meta descriptions, and the headings

It is best practice to squeeze it into the opening paragraph.

Other meta tags

  • Meta keywords attribute:

    A series of keywords you deem relevant to the page in question.

    it helps google and other browsers to know what is the main topic of your site.

  • Meta viewport:

    Gives the browser instructions on how to control the page's dimension and scaling.

  • Meta charset:

    Defines a character set.

  • Meta author:

    Defines the author of a page.

Minify and compress

When you minify your website's css, html and javascript files, you get a little extra page load speed.

Minification is the process of compression code from the original size to the smallest.

The process basically remove every blank space ant try to expose the code in a single line.

It can reduce from 10% - 95% the size of the code.
